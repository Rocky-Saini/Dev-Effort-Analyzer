Terms & Privacy Policy

🔒 Privacy First

We do NOT save any information from your side. All data is stored locally in your browser (localStorage) and never leaves your machine.

Data Storage

  • Project paths are saved in your browser's localStorage only
  • No server-side database exists — zero data persistence on our end
  • Jira story content is processed in-memory and never stored
  • Generated files are saved only in your local repository folder
  • Clearing your browser data removes all saved projects

How It Works

  • The tool runs entirely on your local machine
  • Codebase scanning happens locally via Node.js
  • Kiro CLI runs locally with your authenticated session
  • No code or data is transmitted to external servers by this tool
  • AI processing happens through your local Kiro CLI authentication

Terms of Use

  • This tool is provided as-is for developer productivity
  • Generated documents should be reviewed before sharing
  • AI-generated content may need manual adjustments
  • Users are responsible for the accuracy of shared documents
  • The tool requires Kiro CLI to be installed and authenticated
